# Python Hooks

## PostToolUse

- **Auto-format**: Run `ruff format` after editing `.py` files
- **Lint check**: Run `ruff check` after editing `.py` files
- **Type check**: Run `mypy` or `pyright` after editing `.py` files
- **print() warning**: Warn about `print()` statements — use `logging` module instead

## Scope

Applies to: `**/*.py`, `**/*.pyi`
